Command-based DSC resources

Ready-to-use resources for Windows services, users, groups, scheduled tasks, SQL Server, file systems, XML, JSON, ZIP archives, and POSIX permissions — all invokable from the standard DSC CLI.

Local Configuration Manager

A background service that continuously monitors system state, detects configuration drift, and automatically remediates it — keeping systems compliant around the clock.

{% include ".icons/material/server-network.svg" %}

Modern pull server

The first pull and push server for Microsoft Desired State Configuration with centralized configuration delivery, versioning, and compliance reporting.

{% include ".icons/material/monitor-dashboard.svg" %}

Modern web UI

An admin interface for managing nodes, configurations, compliance status, and role-based access. Don't want to use the UI? OpenDSC also exposes a REST API for programmatic access to all server features.

{% include ".icons/material/file-document-edit-outline.svg" %}

Custom conversion extensions

Ships with extensions that can convert existing PowerShell DSC configuration documents into Microsoft DSC configuration documents formats to facilitate migration from existing DSC workflows.

Open source & AI-ready

Fully open source and built with clear conventions, making it easy to leverage AI to generate new DSC resources in minutes rather than hours.
